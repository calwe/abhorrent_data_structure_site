Records

Records are a way of storing multiple labelled data types in a group. They are statically sized, but can be mutated.

Psuedocode

They can be defined in OCR pseudocode like so

record student
    id : Integer
    name : String
    grade : String
endrecord

The fields can also be read/written to

student1 : student
student1.id = 1
student1.name = "Dave Grohl"
student1.grade = "A*"

print("Student Name: " + student1.name)

Java

In Java, a record is called a class - and can have methods

We can just create a class with no methods to replicate pseudocode records

public class Student {
    public int id;
    public String name;
    public String grade;
}

We can then create and access our class in our main method.

public class App {
    public static void main(String[] args) {
        Student student = new Student();
        Student.id = 1;
        Student.name = "Dave Grohl";
        Student.grade = "A*";

        System.out.println("Student name: " + student.name)
    }
}
